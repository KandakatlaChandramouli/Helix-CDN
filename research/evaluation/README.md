# Helix Runtime Evaluation

This directory contains:
- throughput benchmarks
- scheduler benchmarks
- saturation benchmarks
- queue pressure measurements
- tail latency tracking
- runtime comparison baselines

## Benchmark Categories

### Runtime Integration
Measures adaptive scheduler execution throughput.

### Saturation
Measures overload behavior and queue pressure.

### Comparison
Compares runtime execution against:
- goroutines
- channels

### Ringbuffer
Measures queue runtime overhead.

### Epoll Runtime
Measures kernel event runtime overhead.
